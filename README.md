> **React and TypeScript Starter kit**

## Install

```sh
# Clone this project to your environment
git clone https://github.com/Myoschen/react-typescript-starter.git

# Move to project
cd react-typescript-starter

# Download required dependencies
npm ci
```

## Usage

```sh
# Production mode
npm run build

# Development mode
npm run dev
```
